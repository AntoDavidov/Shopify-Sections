## 🚀 How to Install a Section

1. Go to your Shopify Admin → **Online Store → Themes**
2. Click **Edit code** on your active theme
3. Under `sections/`, click **Add a new section**
4. Paste the `.liquid` file contents and save
5. Go to **Customize** and add the section to any page

## 🧩 Section Details

### AD - Sticky Split Images (`ad-sticky-text-banner.liquid`)
A visually striking section with two side-by-side images and a sticky heading that stays visible as users scroll.

**Settings:**
- Upload separate desktop & mobile images for both left and right sides
- Choose which image shows on mobile (left, right, or both)
- Control section height, text position, heading size & color
